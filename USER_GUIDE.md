## Introduction

The Mental Health Management System is a secure platform designed to help individuals seek mental health support through counseling requests while maintaining privacy and confidentiality.

### Key Features:

- Secure user authentication
- Anonymous or identified counseling requests
- Real-time notifications
- Mental health resources library
- Admin dashboard for request management

## User Features

### Dashboard

Your dashboard provides:

- Overview of your counseling requests
- Quick access to submit new requests
- Unread notifications count
- Mental health tips

### Submit Counseling Request

1. Navigate to "New Request" from dashboard or navbar
2. Fill in the form:
   - **Subject**: Brief description of your concern
   - **Description**: Detailed explanation (be as specific as possible)
   - **Anonymous Option**: Check to submit without revealing identity
3. Click "Submit Request"
4. You'll receive a notification when admin responds

**Privacy Note**: All requests are confidential and only accessible to authorized mental health professionals.

### My Requests

View all your submitted requests with:

- Subject and description
- Current status (Pending, Approved, Completed, Rejected)
- Admin responses
- Submission dates

**Status Meanings**:

- **Pending**: Request is awaiting review
- **Approved**: Request has been reviewed and approved for counseling
- **Completed**: Counseling has been provided
- **Rejected**: Request doesn't meet criteria (with explanation)

### View Request Details

Click "View Details" on any request to see:

- Full description
- Status updates
- Admin response and guidance
- Submission timestamp

### Mental Health Resources

Access helpful articles and tips:

- Browse by category
- Read comprehensive guides
- Learn coping strategies
- Understand mental health conditions

### Submit Feedback

Share your experience:

1. Navigate to "Feedback" in menu
2. Write your message
3. Choose to submit anonymously or identified
4. Click "Submit Feedback"

### User Profile

Manage your account:

- Update full name and email
- Change password
- View account information
- See member since date

### Change Password

1. Go to Profile page
2. Fill in the password change form:
   - Current password
   - New password
   - Confirm new password
3. Click "Change Password"

## Admin Features

### Admin Dashboard

Overview of system statistics:

- Total users count
- Pending requests
- Completed requests
- Published resources
- Recent activity

### Manage Users

View and manage all registered users:

- See user information (name, email, status)
- Activate or block user accounts
- View registration dates
- Monitor user activity

**To Block/Activate User**:

1. Navigate to "Manage Users"
2. Find the user
3. Click status toggle button
4. Confirm action

### Manage Counseling Requests

Review and respond to user requests:

**View All Requests**:

- Filter by status (All, Pending, Approved, Completed, Rejected)
- See user information (unless anonymous)
- View submission dates

**Respond to Request**:

1. Click "Review" on any request
2. Read the user's description carefully
3. Select new status
4. Write your professional response
5. Click "Update Request"
6. User receives notification automatically

**Best Practices**:

- Be empathetic and professional
- Provide actionable guidance
- Recommend resources when applicable
- Follow up on completed requests

### Manage Mental Health Resources

Add and manage helpful content:

**Create New Resource**:

1. Navigate to "Manage Resources"
2. Click "Create New Resource"
3. Fill in:
   - Title
   - Content (detailed article)
   - Category
   - Status (Published/Draft)
4. Click "Create Resource"

**Edit Resource**:

1. Find resource in list
2. Click "Edit"
3. Make changes
4. Update status if needed
5. Save changes

**Delete Resource**:

1. Click "Delete" on resource
2. Confirm deletion

### View User Feedback

Review all user feedback:

- See identified and anonymous feedback
- Identify areas for improvement
- Understand user satisfaction
- Respond to concerns

### Send Notifications

Keep users informed:

- Notifications sent automatically on request updates
- Can create custom notifications
- Users see notifications on dashboard

## FAQ

### General Questions

**Q: Is my information confidential?**
A: Yes, all information is confidential and only accessible to authorized mental health administrators.

**Q: Can I submit requests anonymously?**
A: Yes, check the "Submit anonymously" option when creating a request.

**Q: How long does it take to get a response?**
A: Response times vary, but most requests are reviewed within 24-48 hours.

**Q: Can I delete my account?**
A: Contact the administrator through feedback to request account deactivation.

### Technical Questions

**Q: I forgot my password. What should I do?**
A: Currently, contact the administrator through the feedback form. (Password reset feature can be added)

**Q: Why can't I login?**
A: Ensure:

- Correct email and password
- Account is not blocked
- Cookies are enabled

**Q: The page is not loading correctly**
A: Try:

- Clearing browser cache
- Using a different browser
- Checking internet connection

### Request Questions

**Q: Can I edit my request after submission?**
A: Currently, you cannot edit requests. Submit a new request or contact admin.

**Q: What happens to rejected requests?**
A: You'll receive an explanation from the admin. You can submit a modified request.

**Q: Can I track my request history?**
A: Yes, all your requests are visible in "My Requests" section.
